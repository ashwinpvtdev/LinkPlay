# 🎬 LinkPlay

A modern AI-assisted React-based media player for playing direct media source URLs.

## ✨ Features

- 🎥 MP4 playback
- 🎵 MP3 playback
- 📺 HLS (.m3u8) support
- 📃 Playlist
- 🕒 Recent links
- 💬 Subtitle support (.vtt)
- 📱 Responsive UI
- 🌙 Modern glassmorphism design

## 🛠 Tech Stack

- React
- Vite
- JavaScript
- Tailwind CSS
- hls.js

## 🚀 Run locally

```bash
npm install
npm run dev
```
